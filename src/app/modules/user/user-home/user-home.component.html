สื่อการเรียนการสอน คืออะไร?

สื่อการเรียนการสอน เป็น ตัวกลางหรือช่องทางในการถ่ายทอดองค์ความรู้ ทักษะ ประสบการณ์ จากแหล่ง ความรู้ไปสู่ผู้เรียน และทําให้เกิดการเรียนรู้อย่างมีประสิทธิภาพ ในทางการศึกษา มีคําที่มีความหมายแนวเดียวกันกับสื่อการเรียนการสอน เช่น สื่อการสอน (Instructional Media or Teaching Media), สื่อการศึกษา (Educational Media), อุปกรณ์ ช่วยสอน (Teaching Aids) เป็นต้น

อยากเก่งต้องฝึกฝน
1.สำรวจตัวเองเป็นประจำ

หาเวลาเว้นว่างจากสื่อโซเชียลสัก 1-2 ชั่วโมงมาสำรวจตัวเองสักครั้งในหนึ่งสัปดาห์ ลองถามตัวเองดูว่าขณะนี้คุณมีความสุขกับชีวิตหรือไม่ ได้ทำสิ่งที่ตั้งเป้าหมายไว้จนสำเร็จครบถ้วนทุกประการหรือยัง

2.ตั้งโจทย์ให้ชีวิต

หลังจากที่คุณสำรวจตัวเอง และค้นพบแล้วว่าคุณควรทำอย่างไรกับสิ่งต่างๆ ในชีวิต สิ่งที่คุณควรทำเป็นลำดับถัดไปคือการตั้งโจทย์ให้ชีวิต พูดให้เข้าใจง่ายๆ ก็คือ ตั้งเป้าหมายใหม่ๆ ให้กับชีวิตนั่นแหละ

3.ทดลองทำสิ่งใหม่ๆ

เมื่อตั้งโจทย์ให้ชีวิตเป็นที่เรียบร้อยแล้ว ต่อมา คุณควรลุกขึ้นมาทดลองทำสิ่งใหม่ๆ ดูเสีย เพราะถ้าคุณไม่ลงมือทำอะไรสักอย่าง โจทย์ที่คุณตั้งไว้ก็ไม่มีทางที่จะสำเร็จได้ใช่ไหมล่ะ? ทางเดียวที่คุณจะประสบความสำเร็จได้คือการทดลองทำสิ่งใหม่ๆ เพื่อให้คุณบรรลุเป้าหมายของตัวเอง

4.ให้รางวัลตัวเองอยู่เสมอ

ไม่ว่าใครก็ต้องการกำลังใจในการพัฒนาตนเองต่อไปทั้งนั้น แม้แต่ตัวคุณเองก็เช่นกัน ทุกๆ ครั้งที่คุณทำกระบวนการข้อ 1-2-3 สำเร็จ ขอให้คุณให้รางวัลตัวเองตามสมควร

เคล็ดลับพัฒนาตนเอง
1. เอาชนะความคิดด้านลบในตัวคุณ

ทุกคนล้วนมีความคิดเป็นของตนเอง ความคิดนั้นมีทั้งด้านดีและด้านลบขึ้นอยู่กับสถานการณ์ หากคุณสามารถรู้ทันความคิดด้านลบของคุณแล้วสามารถแปรเปลี่ยนเป็นความคิดดีๆ สิ่งที่ดีๆก็จะเกิดขึ้นกับคุณเสมอ

2. เอาชนะความกลัวอย่างกล้าหาญ

ในโลกใบนี้คงไม่มีใครที่ไม่เคยกลัวอะไรเลย ความกลัวไม่ใช่เรื่องที่เลวร้ายมากขนาดนั้น ความกลัวช่วยให้เรารอดพ้นจากสิ่งชั่วร้ายได้ด้วยการหลีกหนี แต่หากคุณกลัวทุกอย่างคุณก็จะหลีกหนีทุกสิ่งทุกอย่าง

3.มองโลกให้กว้างและตั้งเป้าหมายให้ชัดเจน

โลกเปลี่ยนไปเราก็จะต้องพัฒนาตามให้ทัน อย่าให้ใครมาจำกัดความคิดของเราอยู่แต่ในกะลาแคบๆ ไม่มีอะไรที่เป็นไม่ได้หากเรามีความตั้งใจและพยายามจงเชื่อมั่นว่าคุณสามารถทำอะไรก็ตามให้เกิดขึ้นได้

4. อย่าเสียเวลากับการเปรียบเทียบตนเองกับคนอื่น

การเสียเวลาไปกับการเปรียบเทียบตนเองกับคนอื่นเป็นเรื่องที่คนประสบความสำเร็จไม่ทำกัน เพราะมันไม่ได้มีประโยชน์ที่จะทำให้เกิดการพัฒนาตนเอง เนื่องจากคุณจะไม่สามารถรู้ได้เลยคนเหล่านี้สร้างภาพอะไรมาหลอกคุณ
